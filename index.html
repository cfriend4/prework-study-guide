HTML

  • HTML stands for Hyper Text Markup Language and is the main language used to display conent on a website.
  • An HTML element is a component of an HTML document that tells a web browser how to structure and interpret a part of an HTML document
  • The head element contains meta information about the webpage for the browser.
  • We add links to related files in the head element to tell the browser what is included in our website and how these files should interact.
  • The body element reprsents the visible content shown to the user.
  • A child element is an element nested inside another element.
  • An element refers to an entire element from opening to closing tag.
  • A tag refers only to what appears inside the angle brackets.

CSS

  • CSS is used to define style rules for specific elements of an HTML page and format them.
  • Inline CSS: applies to a specific HTML tag. Useful for quick changes but less flexible than other options because each style must be edited if you make a design change.
  • Internal Style Sheet: Contains style rules for the web page in the head section of the HTML file.
  • External Style Sheet - Preferred in most cases because it allows developers to keep all their CSS rules in a seperate file.
  • When using an external style sheet, we link the file using a link element places in the HTML's head.
  • Within CSS, we can designate a class selector by using .[class name] with styling and then applying it within HTML elements.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch you are currently on
  • git checkout - b branch name: creates a new branch and switches to it
  • git checkout main: switch to main branch
  • git pull origin main: pulls the most up to date version of main to your local device
  • git add -A: commit all files in the working branch
  • git commit -m "add a message": commit a change to the repository and add a message
  • git pull origin main: Pull in the base branch from your github repository
  • git push origin [branch name]: pushes all of our changes

Javascript

  • A variable is a named contained that allows us to store data in our code.
  • Use the const keyword to declare a variable that cannot be changed.
  • The strict equality operator === checks to see if two values are equal, and will return true or false.
  • An array is a single variable that is used to hold a group of data.
  • Arrays can story multiple data types within a single array.
  • In JavasScript and other programing languages, arrays are zero indexed and sequential, meaning that the first item in an array will have index zero and the index of the item will increase by one for each.
  • Use the let keyword to assign a variable that can be reassigned.
  • The .forEach() method can be used as an alternative to a for loop for iterating over arrays.
  • Control flow is the order in which a computer executes code in script. In Javascript it is executed from top to bottom of your file.
  • A function is a set of instructions that tells the computer to do a certain task.
  • Unlike for loops or conditional statements, functions do not automatically execute and must be called by name in order to execute it.
  • Functions make our code more reuseable because we can call it as many times as we need.
  • A function declaration consists of the function keyword follow by the name of the function.
  • To call a function - type the function name followed by parentheses.
