HTML

  • HTML stands for Hyper Text Markup Language and is the main language used to display conent on a website.
  • An HTML element is a component of an HTML document that tells a web browser how to structure and interpret a part of an HTML document
  • The head element contains information about the webpage.
  • The body element reprsents the visible content shown to the user.
  • A child element is an element nested inside another element.
  • An element refers to an entire element from opening to closing tag.
  • A tag refers only to what appears inside the angle brackets.
